# Custom Explanation Docs

This folder is different from `documentation/` (the handover handbook).
`documentation/` explains the whole system at a level meant for a new team
member or the next student group. This folder is for **deep, on-demand
technical dives** into one specific question, written whenever it is asked, at
full depth: every protocol, every constant, every method, every library, with
exact code citations.

Think of `documentation/` as the book, and this folder as an appendix that
grows one chapter at a time as questions come in.

## Docs in this folder

| # | Doc | Question it answers |
|---|-----|----------------------|
| 01 | [GCC to Drone: Connection and Control, In Depth](01_GCC_DRONE_CONNECTION_AND_CONTROL.md) | How does the GCC app connect to and control the system drone? Every protocol, method, and library involved, end to end. |

Ask for the next one any time; it will be added here with the same depth and
the same rule: every claim is checked against the code before it is written
down.
